What I've Made

  • Front-end Application

    Datum is a progressive web app (PWA) created with React.js and the Material-UI library. Multiple key-value-pairs can be grouped and submitted together, then viewed in different modes, currently including "Todos", "Habits", "Timeline", and "Stats". Also features data import/export and offline storage.

  • Database & REST API

    Smartmarks is a bookmark app featuring client-side encryption and tagging. A React.js CRUD front-end requests a REST API written with Express Node.js, storing the bookmarks and associated tags in two MySQL tables. The user login and password never leave the client, so they can be used to encode/decode the data sent to the server.
